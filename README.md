# 每日新聞 Brief

每天整理一份**對手遊產品企劃有價值**的新聞精選，涵蓋台港澳與國際，過濾紅媒與低訊噪比內容。

**公開網頁**：https://timdachung.github.io/news-brief/

---

## 內容結構

- **台港澳** 每日上限 10 則
- **國際** 每日上限 10 則
- 三種相關性標籤：
  - 🎯 **工作直擊**：直接影響手遊產品決策／公司營運
  - 📊 **產業視野**：可拆解學習的案例、競品動態
  - 🌏 **策略背景**：地緣、法規、長期趨勢

## 篩選準則

1. 結構變化 > 單一事件
2. 原始資料 > 二手報導
3. 與工作相關 > 一般熱度
4. 延展性案例 > 熱度爆點

## 避開的來源

- 中國官媒 / 紅媒：人民日報、新華社、環球時報、中國日報、CCTV、CGTN
- 香港親中媒體：大公報、文匯報、香港商報、星島日報
- 俄媒：RT、Sputnik、TASS
- 純政治口水、娛樂八卦、情緒性裁員新聞

## 更新頻率

每日（假日後可能累積統整多天）。每次更新直接覆蓋首頁，歷史透過 git history 追溯。
